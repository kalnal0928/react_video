# Video Player

A cross-platform desktop video player built with Electron, React, and TypeScript.

## Features

- Support for multiple video formats (MP4, AVI, MKV, MOV, WMV, WebM)
- Playlist management
- Video playback controls
- Keyboard shortcuts
- Fullscreen mode

## Development

Install dependencies:
```bash
npm install
```

Run in development mode:
```bash
npm run electron:dev
```

Build for production:
```bash
npm run electron:build
```

## Tech Stack

- Electron
- React 18
- TypeScript
- Vite
- Electron Builder
